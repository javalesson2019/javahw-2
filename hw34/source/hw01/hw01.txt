Домашнее задание

1. Зарегестрироваться на github.com
2. Создать репозиторий с названием javahw - теперь здесь вы будете хранить свои домашние задания по java
3. Создать клон репозитория на домашнем компьютере
4. В репозиторий добавить файл .gitignore, в который доавить содержимое https://github.com/github/gitignore/blob/master/Global/JetBrains.gitignore
5. Сохранить .gitignore добавить изменения, закоммитить и запушить на github
6. В настройках репоизтория включить в качестве контрибютора меня по имени javalesson2019.
   Для этого необходимо перейти на страницу настроек репозитория Settings и в разделе Collaborators указать javalesson2019.
7. Прислать ссылку на mystat на созданный репозиторий

Рекомендую пройти туториал по работе с git: https://githowto.com
